# Dev Patel — Portfolio

Personal developer portfolio built with **Next.js 15**, **Tailwind CSS**, **Framer Motion**, and **GSAP**.

Live at: [dev-portfolio](https://github.com/dev1654/Dev-Portfolio)

---

## Tech Stack

| Layer | Tech |
|---|---|
| Framework | Next.js 15 (App Router) |
| Styling | Tailwind CSS v4 |
| Animations | Framer Motion + GSAP |
| 3D | Three.js |
| Icons | React Icons |

## Sections

- **Hero** — Name, role typewriter, bio, resume link, stats
- **About** — Background and story
- **Skills** — Technologies and tools
- **Experience** — Work history
- **Projects** — Featured builds
- **Education** — Academic background
- **Contact** — Get in touch

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build & Deploy

```bash
npm run build   # production build
npm run start   # production server
```

Recommended deploy: [Vercel](https://vercel.com) — zero config for Next.js.

## Project Structure

```
app/           # Next.js App Router (layout, page, globals.css)
components/    # All section components
public/        # Static assets (resume.pdf, icons)
```
